# Validation Notes

## How to use this package

1. Review `shot-list.md`.
2. Review `timeline-plan.md` and confirm the selected images match the voiceover segments.
3. Tighten the best 2-3 prompts in `image-prompts.md`.
4. Generate only 2-3 images for the first validation.
5. Record usable images, failures, and rule changes here.

## Review questions

- Did Mr.Ai keep cap, mustache, blue clothing, and yellow accent?
- Did each image explain one idea?
- Did each image appear at the right voiceover moment?
- Were B quote segments preserved instead of distorted into new claims?
- Did text need to move to post-production?
- Which layout pattern worked best?
