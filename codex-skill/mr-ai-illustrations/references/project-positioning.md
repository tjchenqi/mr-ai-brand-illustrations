# Project Positioning

## One sentence

Turn AI 闲僧 scripts and AI knowledge topics into consistent Mr.Ai brand illustrations for video explanation scenes.

## What this system is

This is a reusable agent pack for planning and generating Mr.Ai illustration assets. It helps an agent read a口播稿, identify the visual moments, choose the right metaphor, and produce image prompts for 16:9 and 9:16 outputs.

It is designed for:

- Codex
- OpenClaw
- Claude Code or Cursor-like coding agents
- Image generation agents
- Human designers who need a stable brief

## What this system is not

- Not a generic cartoon style prompt.
- Not a PPT infographic generator.
- Not a full Remotion video system.
- Not a replacement for post-production subtitles and overlays.
- Not a clone of the Xiaohei IP.

## Content priority

Prioritize video-internal explanation images that follow the voiceover. A good output should make one abstract AI idea easier to understand in one glance.

Secondary uses:

- Cover image planning
- WeChat article illustration
- Vertical short-video stills
- Transition or chapter visuals

## Required delivery for a script task

Default outputs:

- `shot-list.md`: where each image appears and what it explains.
- `image-prompts.md`: final image prompts, one per image.
- `16x9/`: horizontal output folder.
- `9x16/`: vertical output folder.
- `qa-report.md`: brand and clarity checks.
- `remotion-overlay-notes.md`: optional notes for subtitles, labels, motion, and layout.

If no images are generated yet, still create the planning files and leave output folders ready.
