# Composition Patterns

Use one pattern per image.

## Workflow

Use for input -> process -> output.

Example: Mr.Ai feeds raw notes into a small machine, checks one dial, and receives a clean explanation card.

## Black box reveal

Use for model behavior, hidden reasoning, RAG, evaluation, or debugging.

Example: Mr.Ai lifts a panel on a model box and shines a flashlight on evidence cards inside.

## Before and after

Use for confusion -> clarity, manual -> automated, scattered -> organized.

Example: left side messy prompt scraps, right side clean prompt kit, Mr.Ai bridging the two.

## Risk and guardrail

Use for hallucination, privacy, bias, copyright, overtrust, and data quality.

Example: Mr.Ai places a small yellow guardrail before a red "幻觉" hole.

## Toolchain handoff

Use for multi-agent or multi-tool processes.

Example: Mr.Ai threads a blue line through small labeled tool blocks.

## Evidence stack

Use for trust, verification, source quality, and fact checking.

Example: Mr.Ai stacks evidence cards under a claim until it stands upright.

## Concept machine

Use for abstract but repeatable mechanisms.

Example: a small friendly machine turns "问题" into "步骤" and "输出", with Mr.Ai adjusting the knob.

## Mini comic

Use for a small narrative or misconception correction.

Example: three panels: overtrust, check, corrected answer.

## Originality rule

Do not reuse a metaphor only because it worked before. Start from the current script and invent the simplest physical action that explains the idea.
