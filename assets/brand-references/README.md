# Brand References

These files are Mr.Ai visual references for character consistency:

- `MrAi_logo.png`
- `MrAi_character_ref.png`
- `AI_xianseng_cover.jpg`

Use them to preserve the recurring traits:

- Baseball cap.
- Mustache.
- Rounded middle-aged cartoon face.
- Blue clothing.
- Yellow brand accent.
- Thick dark outline.

Use `MrAi_character_ref.png` as the default image-to-image subject reference. It is intentionally plain: white background, no large logo text, no TM mark, and a centered full-body Mr.Ai pose.

Use `MrAi_logo.png` for brand identity review and as a fallback reference only. Logo files contain text and brand-background signals that can pollute image generation.

Do not use these files as full composition templates. They are character references only.
